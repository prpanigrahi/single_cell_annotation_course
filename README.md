# Single cell data analysis - Annotation
This repo contains codes for doing single cell data annotation of cell types. Covers marker based, reference based, AI assisted annotations.
Youtube playlist for the course: https://www.youtube.com/playlist?list=PLAbpA2ThJEBSG-E1L0K9zwvlJB1vg4gbL

## Marker based approach
- Custom overlap based approach
    - marker_based_annotation/overlap_based_annotation.r
- Statistics based approach
    - Clustermole
    - scType
    - Easycelltype
